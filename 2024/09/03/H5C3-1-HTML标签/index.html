Web标准

主要包括结构、样式、行为。三者相分离。结构写到HTML文件中，样式写到CSS文件中，行为写到JavaScript中。

标签

语法规范

基本语法

  1. 标签是由尖括号包围的关键词
  2. 标签通常成对出现，即双标签。第一个为开始，第二个为结束
  3. 特殊标签：单个标签，例如<br/>，称为单标签

标签关系

双标签关系可分为两类：

  • 包含关系

  • 并列关系

基本结构标签

第一个HTML网页

每个网页都有一个基本的结构标签（骨架标签），页面内容也是在这些基本标签上书写。

HTML页面也称HTML文档。

标签名 定义 说明
<html></html> HTML标签 页面中最大的标签，即 根标签
<head></head> 文档的头部 注意在head标签中，我们必须设置的标签是 title
<title></title> 文档的标题 让页面拥有一个属于自己的网页标题
<body></body> 文档的主体 元素包含文档的所有内容，页面内容 基本是放到body里面的

开发工具

VSCode的使用

  1. 生成页面骨架结构：输入 ! 按下 Tab 键
  2. 利用插件在浏览器中预览页面：单击鼠标右键，在弹出窗口中点击“Open In Default Browser”

更多Tips：

  • 放大缩小视图：Ctrl + 或 Ctrl -

骨架结构新增代码有

  1. <!DOCTYPE>标签
  2. lang 语言
  3. charset 字符集

文档类型声明标签

<!DOCTYPE>文档类型声明，即告诉浏览器使用哪种HTML版本来显示网页

注意：

  1. <!DOCTYPE>声明位于文档中的最前面的位置，处于<html>标签之前。
  2. <!DOCTYPE>不是一个HTML标签，而是文档类型声明标签

lang 语言种类

用来定义当前文档显示的语言

  1. en定义为英语
  2. zh-CN定义为中文

其实对于文档显示来说，定义成某种语言的网页也可以显示别的语言。但是这个定义对浏览器和搜索引擎等还是有作用的

字符集

字符集是多个字符的集合，便于计算机识别和存储各种文字

在<head>标签内，可以通过<meta>标签的charset属性来规定HTML文档应该使用哪种字符编码

charset常用的值有：GB2312, BIG5, GBK 和 UTF-8，其中UTF-8即万国码，基本包含全世界需要用到的字符。

/这是必须要写的代码，否则可能引起乱码的情况。一般情况下，统一使用“UTF-8”编码，尽量写成标准的形式

常用标签

标签语义

根据标签的语义在合适的地方给一个最为合理的标签，可以让页面结构更清晰

标题标签<h1>-<h6>

为了使网页更具有语义化，我们经常在页面中用到标题标签。HTML提供了六个等级的标签。

特点：

  1. 加了标题的文字会加粗，字号依次变大。
  2. 一个标题独占一行。

段落和换行标签

在HTML标签中，<p>标签用于定义段落，它可以将整个网页分为若干个段落

特点：

  1. 文本在一个段落中会根据浏览器窗口的大小自动换行。
  2. 段落和段落之间保有空隙。

如果希望某段文本强制换行，需要使用换行标签<br/>，它是单标签，且知识简单地开始新一行，跟段落不一样，段落之间会插入一些垂直的间距

文本格式化标签

以下语义各有两种标签，更推荐前者，语义更强烈

语义 标签
加粗 <strong></strong>或<b></b>
倾斜 <em></em>或<i></i>
删除线 <del></del>或<s></s>
下划线 <ins></ins>或<u></u>

<div>和<span>标签

无语义，它们只是盒子，用来装内容的。

div，division，表示分割，分区，用于布局，一行只能放一个div，大盒子

span，跨度，用来布局，一行可以多个span，小盒子

图像标签

<img>用于定义HTML页面中的图像

src是<img>标签的必须属性，用于指定图像文件的路径和文件名

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本。图像不能显示的文字
title 文本 提示文本。鼠标放到图像上会显示的文字
width 像素 图像宽度
height 像素 图像高度
border 像素 图像的边框粗细

图像标签属性注意点：

  1. 图像标签可以拥有多个属性，但必须写在标签名的后面
  2. 属性之间不分先后顺序，标签名、属性任意两两之间以空格分开
  3. 采用键值对的格式，即key="value"

路径

<a>用于定义超链接，作用是从一个页面链接到另一个页面（单词anchor，意为锚）

链接的语法格式如下：

属性 作用
href 用于指定链接目标的 url 地址（必须属性），当为标签应用 href 属性时，它就具有超链接的功能
target 用于指定链接页面的打开方式，其中_self 为默认值，_blank为在新窗口的打开方式

而链接有以下分类

  1. 外部链接：例如<a href="http://www.baidu.com">，即BAIDU
  2. 内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可，例如<a href="index.html"> 首页</a>
  3. 空链接：如果当时没有确定链接目标时，<a href="#"> 首页</a>
  4. 下载链接：如果 href 里面地址是一个文件或压缩包，会下载这个文件
  5. 网页元素链接：在网页中的各种网页元素，如文本、图像、表格、音频、视频等都可以添加超链接
  6. 锚点链接：点击链接，可以快速定位到页面中的某个位置
    • 在链接文本的 href 属性中，设置属性值为#名字的形式，如<a href="#two"> episode 2</a>
    • 找到目标位置标签，里面添加一个 id ，其属性=刚才的名字，如：<h3 id="two"> introduction</h3>

相对路径

我们都知道.代表当前目录，..代表上级目录，那么如何表示上级的上级目录呢？答案是../..。

表格标签

基本语法

表格的基本语法如下：

  1. <table></table>用于定义表格
  2. <tr></tr>用于定义表格中的行，必须嵌套在<table></table>
  3. <td></td>用于定义表格中的单元格，必须嵌套在<tr></tr>中。字母 td 指表格数据，即数据单元格的内容

表头单元格

一般表头单元格位于表格的第一行或第一列，表头单元格里面的文本加粗居中显示。

<th>表示HTML表格中的表头部分（table head）

表格属性

这部分开发其实不常用，后面通过CSS设置。注意点只有：

  1. 记住这些英文单词，CSS会使用
  2. 直观感受表格的外观形态

表格结构标签

使用场景：表格可能很长，为了更好表示语义，可以将表格分割为 表格头部和表格主题两大部分。

在表格标签中，分别用：

  • <thead>表示表格的头部区域，内部必须拥有<tr>标签
  • <tbody>表示表格的主体区域

以上标签都应放在<table></table>标签中

合并单元格

最简单的合并单元格方法

  • 跨行合并：rowspan="合并单元格的个数"
  • 跨列合并：colspan="合并单元格的个数"

目标单元选中后，根据跨越方向进行合并（上到下，左到右）

总的来说，合并单元格三部曲：

  1. 先确定跨行还是跨列合并
  2. 找到目标单元格，写上合并方式=合并的单元格数量
  3. 删除多余的单元格

表格总结

  1. 表格的相关标签

    table, tr, th, td, thead, tbody

  2. 表格的相关属性

    align, width, height, cellspacing, cellpadding

  3. 合并单元格

    跨行：rowspan

    跨列：colspan

列表标签

列表分为三大类：无序列表、有序列表和自定义列表

无序列表

<ul>表示无序列表，一般以项目符号呈现列表项。而列表项使用<li>定义

无序列表的基本语法格式如下：

  • <ul></ul>之间只能嵌套<li></li>，不能在其中输入其他标签或文字
  • <li></li>之间相当于一个容器，可以容纳所有元素
  • 无序列表会带有自己的样式属性，但在实际使用时，会使用CSS设置

有序列表

<ol>和<li>，基本规则同上

自定义列表

常用于对属于或名词进行解释和描述，定义列表的列表项前没有任何项目符号

<dl>用于定义描述列表，与<dt>（定义项目/名字）和<dd>（描述每一个项目/名字）一起使用

其基本语法如下：

  1. <dl>中只能包含<dt>,<dd>
  2. dt和dd个数没有限制，经常是一个前者对应多个后者

表单标签

使用表单目的是为了收集用户信息

在HTML中，一个完整的表单通常由表单域、表单控件（即表单元素）和提示信息 3 个部分构成

表单域

即包含表单元素的区域。

<form>用于定义表单域，以实现用户信息的收集和传递。<form>会把它范围内的表单元素信息提交给服务器

常用属性：

属性 属性值 作用
action utl地址 用于指定接收并处理表单数据的服务器程序的url地址
method get/post 用于设置表单数据的提交方式，其取值为get或post
name 名称 用于指定表单的名称，以区分同一个页面的多个表单域

表单控件/表单元素

即允许用户在表单中输入或者选择的内容控件

  1. input输入表单元素
  2. select下拉表单元素
  3. textarea文本域元素

这些表单元素都应包含在form表单域中，并有name属性

<input />表单元素

用于收集用户信息，<input />为单标签

该标签中包含多种属性，其中非常重要的是type属性，根据不同的type属性值，输入字段拥有很多种形式（文本字段、复选框、掩码后的文本控件、单选按钮、按钮等）

type 属性包括：

除此之外，常用属性还有：

  1. name 和 value 是每个表单元素都有的属性值，主要给后台人员使用

  2. name 是表单元素的名字，要求单选按钮和复选框有相同的name值。name属性可用于区别不同的表单。如果radio或checkbox是一组，我们必须给他们命名相同的名字。

  1. checked属性主要针对单选按钮和复选框，一打开页面，就要能默认选中某个表单元素。特别地，若想让某单选按钮或复选框是选中状态，只需把checked属性的value设置为”checked”

Q: 有些表单元素想刚打开页面就默认显示某些文本，怎么做？

A: 可以给这些表单元素设置value属性=值

<label>标签

为input元素定义标注，用于绑定一个表单元素，当点击<label>标签内的文本时，浏览器会自动将焦点（光标）转到或者选择对应的表单元素上，用来增加用户体验

最重要的是：<label>标签的 for 属性应当与相关元素的 id 属性相同

<select>表单元素

定义下拉列表

在<option>中定义selected="selected"，当前项即为默认选中项

<textarea>表单元素

用于定义多行文行输入的控件，可以输入更多的文字，常见于留言板、评论

属性rows和cols在实际开发中不会使用，都是用CSS来改变大小的

心得

  1. 表格标签默认无边框，可以用来将内容排列齐整
