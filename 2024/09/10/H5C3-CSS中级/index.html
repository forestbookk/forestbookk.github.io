CSS的复合选择器

CSS中，选择器的类型分为基础选择器和复合选择器。复合选择器是建立在基本选择器上，进行组合形成。

常用的复合选择器包括：后代选择器、子选择器、并集选择器、伪类选择器等等。

  • 元素二可以是儿子，也可以是孙子，只需是元素一的后代
  • 元素一、元素二可以是任意基础选择器

子选择器

只能选择某元素的最近一级子元素

并集选择器

可选择多组标签，同时为他们定义相同的格式。通常用于集体声明。

任何形式的选择器都可成为并集选择器的一部分。

伪类选择器

用于向某些选择器添加特殊的效果，比如给链接添加特殊效果，若选择第一个、第n个元素。

书写最大的特点是冒号。

伪类选择器种类很多，有链接伪类、结构伪类等。此处讲解常用的链接伪类选择器。

链接伪类选择器

LVHA的循顺序声明：:link - :visited - :hover - :active

:link表示鼠标点击之前，也称为原始状态 :hover表示鼠标悬停状态 :active表示鼠标点击状态 :visited表示鼠标点击之后状态

由于 a 链接在浏览器中具有默认样式，所以实际工作中，需要为链接单独指定格式

:focus 伪类选择器

用于选取获得焦点的表单元素。焦点，即光标。一般<input>类表单元素才能获取。因此这个选择器主要针对于表单元素。

效果为点击后，input元素变色。

元素显示模式

概念

元素（标签）以什么方式进行显示。如：<div>自己占一行；一行可显示多个<span>。

HTML元素一般分为块元素和行内元素两种类型。

块元素

常见块元素：<h1>~<h6>、<p>、<div>、<ul>、<ol>、li等。

块元素的特点是：

  1. 独占一行
  2. 高、宽、外边距、内边距可控
  3. 宽度默认是容器（父级宽度）的100%。
  4. 是一个容器/盒子，里面可以放行内或块级元素

注意：

  • 文字类的元素内不能使用块级元素
  • <p>标签主要用于存放文字，因此<p>内不能放块级元素，尤其不能放<div>
  • 同理，标题类也是文字类块级标签，里面也不能放其他块级元素

行内元素

常见的行内元素有<a>,<strong>,<b>,<em>,<i>,<del>,<span>等。

行内元素也称内联元素。

行内元素的特点：

  1. 相邻行内元素在一行上，一行可以显示多个
  2. 高、宽直接设置是无效的
  3. 默认宽度就是它本身内容的宽度
  4. 行内元素只能容纳文本或其他行内元素

注意：

  • 链接中不能再放链接
  • 特殊情况下，链接中可以放块级元素，但是给<a>转换一下块级模式最安全

行内块元素

特殊的标签，如<img/>, input</>, <td>，它们同时具有块元素和行元素的特点。

行内块元素的特点：

  1. 和相邻行内元素（行内块）在一行上，但是之间有空白缝隙，一行可以显示多个
  2. 默认宽度就是它本身内容的宽度
  3. 高度、行高、外边距及内边距可控（块元素特点）

模式总结

元素模式 元素排列 设置样式
块级元素 一行只能放一个 可设置宽度高度
行内元素 一行多个 不可直接设置宽度高度
行内块元素 一行多个 可设置宽度高度

模式转换

特殊情况下，需要元素模式的转换，比如想要增加链接<a>的触发范围。

  • 转换为块元素：display:block
  • 转换为行内元素：display:inline
  • 转换为行内块：display:inline-block

背景

背景属性可以设置背景颜色、背景图片、背景平铺、背景图片位置、背景图像固定等

背景颜色

background-color，默认值是transparent（透明）。

背景图片

background-image，实际开发常见于logo和装饰性的小图或超大背景图。

优点：便于控制位置

参数值 作用
none 无背景图
url 使用绝对或相对地址指定背景图像
背景图片后面的地址，必须加UTL，同时路径不能加引号

背景平铺

background-repeat

参数值 作用
repeat 背景图在纵向和横向上平铺（默认）
no-repeat 背景图像不平铺
repeat-x 横向平铺
repeat-y 纵向平铺

背景图片位置

background-position: x y;

x坐标和y坐标，可以使用方位名词或精确单位

参数值 说明
length 百分数 \ 由浮点数子和单位标识符组成的长度值
position top \ center \ bottom \ left \ right

  1. 参数是方位名词：

    • 如果指定的两个值都是方位名词，则前后顺序无关
    • 如果只指定一个，另一个省略，缺省值为居中

  2. 参数是精确单位：

    • 都是精确坐标：1是x，2是y
    • 如果只指定一个，该数值必定是x，另一个默认垂直居中

  3. 参数是混合单位

    • 第一个是x，第二个是y

背景图像固定（背景附着）

background-attachment：背景图像是否固定或随着页面的其余部分滚动

后期可以制作视差滚动的效果

参数 作用
scroll 随对象内容滚动
fixed 图像固定

背景复合写法

可以合并简写在同一个属性background中。

没有特定的书写顺序，一般约定为：

背景颜色 背景颜色地址 背景平铺 背景图片位置

背景色半透明

  • 最后一个参数是alpha透明度，取值在0~1之间
  • 习惯省略0.3的0，即写为background: rgba(0,0,0,.3)
  • 背景半透明指盒子背景半透明，盒子内容不受影响
  • CSS3新增属性，IE9+浏览器才支持。但是实际开发中，不太关注兼容性写法了，可以放心使用
